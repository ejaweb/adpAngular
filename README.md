# ADP Angular
ADP Angular Assessment

### Setup & Install
`npm run setup`

### Build & View / Test
`gulp`

### Technologies Used
- Angular JS
 - Enterprise structure
 - Modular & Extendable
 - Uses ui-router
- Gulp Tasks
 - SCSS Compression
 - JS Uglification + Babel ES6
 - Development and Production Tasks
 - Angular Template Caching
 - Minification of Assets
- BrowserSync
 - No server setup required
 - Faster development
 - Live reloads on file changes
